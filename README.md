# Product Management Admin Dashboard

A modern, responsive admin dashboard for managing products, built with **React**, **TypeScript**, and **Tailwind CSS**. The project focuses on clean component architecture, reusable UI, and modern React state management practices.

##  Features

* Create, edit, and delete products
* Delete confirmation modal
* Live product search
* Category filtering
* Responsive layout (mobile cards and desktop table)
* Reusable modal and form components
* Dynamic stock status badges
* Clean and scalable component structure

##  Tech Stack

* React
* TypeScript
* Tailwind CSS
* Next.js

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.
